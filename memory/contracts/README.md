# Memory Contracts

This directory contains reusable declarative contracts for memory object families and memory operations. Runtime memory tools should implement these contracts later without changing agent-facing semantics.

## Files

- [object-family-contract.yaml](object-family-contract.yaml): reusable contract template for a new memory object family.
- [access-operation-contract.yaml](access-operation-contract.yaml): operation request/response contract template.
- [retention-contract.yaml](retention-contract.yaml): retention, archival, expiration, and deletion template.
