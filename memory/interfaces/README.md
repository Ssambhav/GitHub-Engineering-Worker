# Memory Interfaces

This directory defines future runtime interface boundaries. These are not implementations.

## Interfaces

- [memory-service.interface.yaml](memory-service.interface.yaml): service capability contract.
- [memory-store.interface.yaml](memory-store.interface.yaml): storage boundary contract.
- [memory-query.interface.yaml](memory-query.interface.yaml): query and retrieval boundary contract.
- [memory-validator.interface.yaml](memory-validator.interface.yaml): validation boundary contract.
